# 💳 信用卡活动追踪系统

## 📁 文件结构

```
campaigns/
├── README.md                      # 本说明文件
├── campaigns.csv                  # 活动总览（所有活动的索引）
├── C202501_rakuten_ribo.csv      # 具体活动详情
├── C202502_xxx.csv               # 未来活动...
└── templates/
    └── campaign_template.csv      # 活动模板
```

## 📋 使用方法

### 1. 查看所有活动

打开 `campaigns.csv` 查看所有活动的概览。

### 2. 添加新活动

1. 复制 `templates/campaign_template.csv`
2. 重命名为 `CYYYYMM_活动简称.csv`（如 `C202511_ana_campaign.csv`）
3. 填写详细信息
4. 在 `campaigns.csv` 中添加索引记录

### 3. 更新活动状态

直接编辑对应的活动CSV文件，更新：
- 条件完成状态
- 操作记录
- 活动状态

### 4. 活动完成后

1. 在活动CSV中标记"已完成"
2. 在主账本（如 `2025/10.csv`）记录收益：
   ```csv
   2026-02-XX,收入,投资收益,4000,JPY,楽天ポイント,,リボ活动返点C202501
   ```
3. 在 `campaigns.csv` 中更新状态为"已完成"
