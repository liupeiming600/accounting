# 记账规则与偏好

- 币种：仅使用 JPY、RMB；金额为正数，默认单位为分隔符中的币种。
- 账户：常见账户包括 现金、Suica、Paseli、Amex Gold、ANA Pay、JAL Pay、WAON、乐天Pay、三井NL/Olive/master 等；Paseli/ANA Pay/JAL Pay/WAON 视为电子钱包。
- 转账处理：账户之间的资金划转记为“转账”，作为中性项目，不计入收支汇总（可保留为两行：转出与转入，或单行带备注）。
- 记录粒度：每条流水一行；`日期` 使用交易发生日。
- CSV：UTF-8 with BOM（utf-8-sig），表头固定为：`日期, 类型, 分类, 金额, 币种, 账户, 商户, 备注`。
- 备注：用于补充活动名/订单号/对账说明。
- 月度文件：每月一个 `MM.csv`；如需统计可生成 `summary.md` 与 `records.csv`。

清洗与规范化建议
- 分类与账户统一命名，避免同义多写（如 PayPay/Paypay 统一为 PayPay）。
- 对“便利店/日用品/外食/交通费/光热费/房租/娱乐/医疗/衣服/书籍/旅游/游戏”等常见分类保持一致。
- 异常值（特别大的金额）需人工复核。
- 仅接受币种集合 {JPY, RMB}，其余需转换或标注汇率。
